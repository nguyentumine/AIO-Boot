WinContig es una herramienta de desfragmentación stand-alone muy fácil de usar que no crea ningún directorio de instalación ni modifica de ninguna forma las entradas del Registro de Windows en su computadora. Su propósito es desfragmentar solamente los archivos sin la necesidad de desfragmentar el disco completo. Además, WinContig le permite agrupar los archivos en perfiles, y también acepta un gran número de definiciones opcionales en línea-de-comando opcionales que usted puede usar si lo desea para controlar la forma en que opera el programa. 

WinContig es una aplicación diseñada para sistemas operativos Windows XP, Windows Vista, Windows 7, Windows 8/8.1, Windows 10. 

Como un usuario de Windows XP, Windows Vista, Windows 7, Windows 8/8.1 o Windows 10 usted tiene que haber hecho log-in con derechos totales como administrador.

==================================================================== 
Instalación
====================================================================
WinContig es un archivo ejecutable stand-alone, y no requiere ningún proceso de instalación ni DLLs adicionales. Para comenzar a usarlo, solamente descárguelo, extraiga el contenido del archivo ZIP a la carpeta que usted prefiera, y ejecútelo.

NOTA: Antes de instalar la versión actual de WinContig, usted tiene que eliminar cualquier versión anterior de este programa. También, en la primera ocasión que se ejecute este programa, se configurarán los valores a sus condiciones predefinidas.

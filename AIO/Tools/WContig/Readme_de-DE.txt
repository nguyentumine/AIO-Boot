WinContig ist ein einfach zu bedienendes, eigenständiges Defragmentierungwerkzeug, das keine Installationsverzeichnisse oder Registry-Einträge auf dem Computer erstellt. Damit können einzelne Dateien schnell defragmentiert werden, ohne den gesamten Datenträger defragmentieren zu müssen. Darüber hinaus ermöglicht WinContig, Dateien in Profile zu gruppieren und akzeptiert eine Reihe von optionalen Befehlszeilenoptionen für die Programmsteuerung.

WinContig läuft auf Windows XP, Windows Vista, Windows 7, Windows 8/8.1 und Windows 10 Betriebssysteme.

Benutzer von Windows XP, Windows Vista, Windows 7, Windows 8/8.1 oder Windows 10 müssen über volle Administratorrechte verfügen.

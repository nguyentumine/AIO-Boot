WinContig est un outil de défragmentation, autonome et facile à utiliser, qui ne crée pas de répertoire d'installation ni d'éléments dans le Registre sur votre ordinateur. Son but est de défragmenter rapidement des fichiers sans avoir besoin de défragmenter le disque entier. En outre, WinContig vous permet de grouper des fichiers dans des profils, et il accepte aussi un certain nombre de commutateurs de ligne de commande facultatifs que vous pouvez utiliser pour contrôler la façon dont le programme fonctionne.

WinContig est une application conçue pour les systèmes d'exploitation Windows XP, Windows Vista, Windows Seven, Windows 8/8.1, Windows 10. 

En tant qu'utilisateur de Windows XP, Windows Vista, Windows Seven, Windows 8/8.1 ou Windows 10 vous devez avoir ouvert une session avec les pleins droits d'administration.

==================================================================== 
Installation
====================================================================
WinContig est un exécutable autonome, et il ne nécessite pas de processus d'installation ou de DLLs supplémentaires. Pour commencer à l'utiliser, téléchargez-le simplement, extrayez le contenu de l'archive ZIP dans un dossier quelconque, puis exécutez-le.

NOTE: Avant d'installer la version actuelle de WinContig, vous devez supprimer l'ancienne version. En outre, la toute première fois où il est exécuté, il définira tous les paramètres aux valeurs par défaut.
